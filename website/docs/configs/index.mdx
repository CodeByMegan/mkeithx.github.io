This section covers the custom configuration powering the **MKX SpaceHub**™ website.
From its initial GitHub commit, the **MKX SpaceHub**™ has been organized as a monorepo with [examples](https://github.com/mkeithX/mkeithx.github.io/tree/main/examples) and built with [Docusaurus TypeScript](https://docusaurus.io/docs/typescript-support) variant along with some awesome tools like [VS Code](https://code.visualstudio.com/).


## Installation

```bash
npx create-docusaurus@latest website classic --typescript
```


```bash npm2yarn
npm run start
```

## Troubleshooting
Resolve issues caused by mismatching cached dependencies - especially during upgrades.

### Step one

Clear all generated assets, caches and build artifacts:

```bash npm2yarn
npm run clear
```
### Step two

Remove `node_modules` and the lock file(s):

<Tabs>

  <TabItem value="bash" label="Bash" default>

```bash
rm -rf node_modules yarn.lock package-lock.json
```


  </TabItem>
  <TabItem value="pwsh" label="PowerShell">

```sh
@('node_modules','yarn.lock','package-lock.json') | Remove-Item -Force -Recurse -ErrorAction SilentlyContinue
```
  </TabItem>

</Tabs>

### Step three
Reinstall packages:

```bash npm2yarn
npm install
```
