# Configuration {#website-configuration}

This section covers the custom configuration powering the **MKX SpaceHub**™ website.
From its initial GitHub commit, the **MKX SpaceHub**™ has been organized as a monorepo with [examples](https://github.com/mkeithX/mkeithx.github.io/tree/main/examples) and built with [Docusaurus TypeScript](https://docusaurus.io/docs/typescript-support) variant along with some awesome tools like [VS Code](https://code.visualstudio.com/).


## Installation

```bash
npx create-docusaurus@latest website classic --typescript
```


```bash npm2yarn
npm run start
```
