In a TypeScript Docusaurus codebase, the configuration file is named `docusaurus.config.ts`. You can see the [custom configuration](https://github.com/mkeithX/mkeithx.github.io/blob/main/website/docusaurus.config.ts) of this website.

```ts title="docusaurus.config.ts"
import type {Config} from '@docusaurus/types';
import type * as Preset from '@docusaurus/preset-classic';

const config: Config = {
  title: 'My Site',
  favicon: 'img/favicon.ico',

  /* Your site config here */

  presets: [
    [
      'classic',
      {
        /* Your preset config here */
      } satisfies Preset.Options,
    ],
  ],

  themeConfig: {
    /* Your theme config here */
  } satisfies Preset.ThemeConfig,
};

export default config;
```

## Troubleshoot
Resolve issues caused by mismatching cached dependencies - especially during upgrades.

### Step one

Clear all generated assets, caches and build artifacts:

```bash npm2yarn
npm run clear
```
### Step two

Remove `node_modules` and the lock file(s):

<Tabs>

  <TabItem value="bash" label="Bash" default>

```bash
rm -rf node_modules yarn.lock package-lock.json
```


  </TabItem>
  <TabItem value="pwsh" label="PowerShell">

```sh
@('node_modules','yarn.lock','package-lock.json') | Remove-Item -Force -Recurse -ErrorAction SilentlyContinue
```
  </TabItem>

</Tabs>

### Step three
Reinstall packages:

```bash npm2yarn
npm install
```
