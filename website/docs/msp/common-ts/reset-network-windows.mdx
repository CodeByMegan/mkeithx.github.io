# Reset Network: Windows 10 and 11

If you're experiencing network connectivity issues on your Windows 10 or Windows 11 PC, resetting network settings can often resolve the problem. This process will revert your network configuration to its default state, potentially fixing any misconfigurations or conflicts. Here's how to reset network settings on both Windows 10 and Windows 11:

## Via Settings

<Tabs groupId="operating-systems">
  <TabItem value="win10" label="Windows 10">

1. **Open Settings**: Click on the Start menu, then select "Settings" (the gear icon).

2. **Navigate to Network & Internet**: In the Settings window, click on "Network & Internet."

3. **Network Reset**: Scroll down on the left sidebar and select "Network Reset."

4. **Reset Network**: Click on the "Reset now" button to initiate the network reset process.

5. **Confirmation**: A confirmation dialog will appear. Click on "Yes" to confirm and proceed with the network reset.

6. **Restart Your PC**: After the network reset is complete, restart your computer for the changes to take effect.

  </TabItem>
  <TabItem value="win11" label="Windows 11">

1. **Open Settings**: Click on the Start menu, then select "Settings" (the gear icon).

2. **Navigate to Network & internet**: In the Settings window, click on "Network & internet."

3. **Advanced Network Settings**: Scroll down and click on "Advanced network settings."

4. **Network Reset**: Under the Network reset section, click on "Reset now."

5. **Confirmation**: A confirmation dialog will appear. Click on "Yes" to confirm and proceed with the network reset.

6. **Restart Your PC**: After the network reset is complete, restart your computer for the changes to take effect.
  </TabItem>
</Tabs>


## Via Command Prompt

1. **Open Command Prompt**: Press Win + R keys to open the Run dialog, type cmd, and press Enter.

2. **Run Command**: In the Command Prompt window, type the following command and press Enter:

```sh
ipconfig /release
```

```sh
ipconfig /flushdns
```
```sh
ipconfig /renew
```

**Optional**

```sh
netsh int ip reset
```

```sh
netsh winsock reset
```



## Additional Notes

:::tip[Important]



- **Backup**: Before resetting your network settings, ensure that you have a backup of any important network configurations or settings.
  
- **Driver Updates**: After resetting network settings, you may need to update your network adapter drivers to ensure optimal performance.

- **Wi-Fi Passwords**: You'll need to re-enter Wi-Fi passwords and reconfigure any custom network settings after performing a network reset.



:::

By following these steps, you can easily reset network settings on both Windows 10 and Windows 11, helping to resolve network connectivity issues and restoring optimal network performance.
