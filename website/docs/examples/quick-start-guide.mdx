# Quick Start Guide

## Step 1: Installing Python

1. **Download Python**:
   - Go to the [Python website](https://www.python.org/downloads/) and download the latest version.

2. **Run the Installer**:
   - For Windows, check "Add Python to PATH" during installation.
   - For macOS, use Homebrew:
     ```bash
     brew install python
     ```

3. **Verify Installation**:
   ```bash
   python --version
   ```

## Step 2: Setting Up a Virtual Environment

1. **Create Virtual Environment**:
   ```bash
   python -m venv myenv
   ```

2. **Activate the Virtual Environment**:
   - **Windows**:
     ```bash
     myenv\Scripts\activate
     ```
   - **macOS/Linux**:
     ```bash
     source myenv/bin/activate
     ```

## Step 3: Installing Django

1. **Install Django**:
   ```bash
   pip install django
   ```

2. **Verify Installation**:
   ```bash
   python -m django --version
   ```

## Step 4: Creating a Django Project

1. **Start a New Project**:
   ```bash
   django-admin startproject myproject
   ```

2. **Navigate to Your Project**:
   ```bash
   cd myproject
   ```

3. **Run the Development Server**:
   ```bash
   python manage.py runserver
   ```
   - Access your project at `http://127.0.0.1:8000/`.

## Conclusion

You've successfully installed Python and Django, and set up your first project. Happy coding!
