# Fix Cloud PC Issues



This article will cover some common Cloud PC problems and provide effective solutions to help you get back on track.

<CloudPC className="logoBadge"/>


## **Connectivity Issue**

<BrowserWindow url="/Connectivity">

If you’re having trouble connecting, try these steps:

- **Check Your Internet Connection:** Make sure your internet is stable and working properly.
- **Restart the Application:** Close the Cloud PC app or your web browser, then reopen it.
- **Clear Your Browser Cache:** If you’re using a browser, clearing the cache and cookies can help.

## **Latency Issues**

<BrowserWindow url="/Latency">

If your Cloud PC feels sluggish, here’s what you can do:

- **Close Unnecessary Applications:** Try to limit the number of apps you have open at the same time.
- **Check Resource Usage:** Look at your [CPU and memory usage](#monitor-uptime) to see if any apps are using too many resources.
- **Restart Your Cloud PC:** A simple [restart can refresh](#restart) the system and improve performance.

### **Check for Updates**
Keeping your system and app up-to-date comes latest features and security improvements. You'll notice if update is available if there's a green arrow on your Cloud PC client.

![update cpc](/img/guides/cpc-update.png)

### **Monitor Uptime**
You can check uptime in Task Manager (Ctrl + Shift + Esc), displayed as [**DAYS: HOURS: MINUTES: SECONDS**] under the "Performance" tab. Aim to restart your Cloud PC at least once a week to refresh system resources and apply updates.

![alt text](/img/guides/pc-uptime.png)

### **Restart**
Navigate to [windows365.microsoft.com](https://windows365.microsoft.com/) to see the Cloud PCs they have access to.

![close-ms-teams](/img/troubleshooting/cloud-pc.png)
