# Fix Network Issues


:::caution[important]
Elevated access required. Contact your administrator for further assist.

:::


### Step 1: Release IP Address
```bash
ipconfig /release
```

### Step 2: Flush DNS Cache
```bash
ipconfig /flushdns
```

### Step 3: Renew IP Address
```bash
ipconfig /renew
```

### Step 4: Reset TCP/IP Stack
```bash
netsh int ip reset
```

### Step 5: Reset Winsock Catalog
```bash
netsh winsock reset
```
### Step 6: Restart Computer

```bash
Restart-Computer
```
