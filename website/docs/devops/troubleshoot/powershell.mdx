# PowerShell

Learn about different editions of PowerShell and explore installation methods for various platforms.

## Edition

To determine the current version and edition of PowerShell, run:

```sh
$PSVersionTable
```

## Installation Methods

<Tabs>
  <TabItem value="winget" label="Winget (Recommended)" default>

```bash
winget search Microsoft.PowerShell
winget install --id Microsoft.Powershell --source winget
```

  </TabItem>

  <TabItem value="msi" label="MSI Installer">

For MSI installation, visit the [GitHub Page](https://github.com/PowerShell/PowerShell/releases) for downloads.

  </TabItem>

  <TabItem value="dotnet" label=".NET Global Tool">

Install PowerShell as a [.NET Global tool](https://dotnet.microsoft.com/download) if you have the .NET Core SDK:

```bash
dotnet tool install --global PowerShell
```

</TabItem>

<TabItem value="other" label="Other Platforms">

For Linux and macOS, refer to the installation guides:

- [Linux Installation Guide](https://docs.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-linux?view=powershell-7.4)
- [macOS Installation Guide](https://docs.microsoft.com/en-us/powershell/scripting/install/installing-powershell-on-macos?view=powershell-7.4)

</TabItem>
</Tabs>

:::caution Important

When upgrading PowerShell, use the same installation method as initially used.

:::
