# Fix common issues in MS Teams

### Clear cache


1. **Close Teams:** If it's still open, right-click the Teams icon on the taskbar and select **Quit**.

2. **Open Run Dialog:** Press **Windows logo key + R**.

![Close Teams](/img/troubleshooting/close-ms-teams.png)
