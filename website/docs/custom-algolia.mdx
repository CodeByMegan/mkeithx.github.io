# Overview

Docusaurus V3 is a powerful tool for creating static websites with ease, but one feature it lacks out of the box is search functionality. Fortunately, integrating Algolia search into your Docusaurus site is a straightforward process that greatly enhances the user experience. 

In this article, outlines steps to add Algolia search to your Docusaurus V3 site.




## Prerequisites

Before we begin, make sure you have the following prerequisites:

- A Docusaurus V3 site set up and running.
- An Algolia account. You can sign up for a free account on the [Algolia website](https://www.algolia.com/).


## Configuration

### Default 

Add the following Algolia search configuration:

```js title="docusaurus.config.js"
module.exports = {
  ...
  algolia: {
    apiKey: 'YOUR_API_KEY',
    indexName: 'YOUR_INDEX_NAME',
    appId: 'YOUR_APP_ID',
  },
  ...
};
```

### Custom

In this method, we will extend Algolia search React component by utilizing the [Swizzle](https://docusaurus.io/docs/swizzling) method:


Add the following on your site configuration:

```tsx title="docusaurus.config.ts"
import type { Config } from "@docusaurus/types";
import type * as Preset from "@docusaurus/preset-classic";

const config: Config = {
  title: "My Site",
  favicon: "img/favicon.ico",

  // highlight-start
  scripts: [
    {
      src: "https://YOUR_APP_ID-dsn.algolia.net",
      rel: "preconnect",
      crossorigin: true,
    },
  ],
  // highlight-end

  /* Your site config here */
};

export default config;
```


**JavaScript**

```bash npm2yarn
npm run swizzle @docusaurus/theme-classic SearchBar
```
**Typescript**

```bash npm2yarn
npm run swizzle @docusaurus/theme-classic SearchBar --typescript
```

Choose the ``--wrap`` option and add the following configuration in your ``src/theme/SearchBar``:

}
