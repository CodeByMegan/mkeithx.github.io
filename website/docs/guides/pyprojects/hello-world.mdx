### Overview

:::tip[What's this?]

In this guide, we're going to find out how print "Hello, world" in 10 various ways using python.
:::



1. Funtion:

```py
print("Hello, world!")
```

2. Variable:

```py
msg = "Hello, world!"
print(msg)
```

3. String Concatenation

```py
greeting = "Hello, "
subject = "World!"
print(greeting + subject)
```

4. Using an f-string (formatted string literal):

```py
greeting = "Hello, "
subject = "World!"
print(f"{greeting}{subject}")
```

5. Multiple `print` statements:

```py
print("Hello,", end=" ")
print("World!")
```

6. The `sys.stdout.write` method:

```py
import sys
sys.stdout.write("Hello, World!\n")
```

7. The `sys.out.write` with `format`:

```py
import sys
sys.stdout.write("{}\n".format("Hello, World!"))
```

8. Using `join` method:

```py
message_parts = ["Hello," , "World!"]
print(''.join(message_parts))
```

9. Using `str.format` method:

```py
greeting = "Hello, {}!"
subject = "World"
print(greeting.format(subject))
```

10. The `repr` function:

```py
print(repr("Hello, World!"))
```
