

# Code Snippets

Here are a few Python code snippets to help you get familiar with the language:

1. Variables and Data Types

```py
# Variables and Data Types
name = "John"
age = 30
is_student = False
```

2. Lists

```py
## List:
fruits = ['apple', 'banana', 'orange']

for fruit in fruits:
    print(fruit)
```

3. Loops

```py
# Loops
for fruit in fruits:
    print(fruit)
```

4. Functions

```py
# Functions
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Output: Hello, Alice!
```

5. Control Flow Statements

#### If-Else Statement

```py
# If-Else Statement
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")
```

#### While Loop

```py
# While Loop
count = 0
while count < 5:
    print(count)
    count += 1
```

#### For Loop

```py
# For Loop
for i in range(5):
    print(i)
```
