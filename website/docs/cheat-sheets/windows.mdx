## Network

<details>
<summary>Network reset</summary>

```bash
ipconfig /release
```
```bash
ipconfig /flushdns
```
```bash
ipconfig /renew
```
:::caution[Optional]

Continue these two commands if you want to reset your network completely.

:::


```bash
netsh int ip reset
```
```bash
netsh winsock reset
```
Restart computer.

</details>

<details>
<summary>Reports</summary>

```bash
netsh interface show interface
```
```bash
netsh wlan show wlanreport
```
```bash
netsh interface ip show dnsservers
```

</details>
