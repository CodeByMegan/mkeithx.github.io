# Welcome
**MKX SpaceHub** is a simple documentation website that features guides, examples, code snippets, etc. for basic software development and infrastructure management.

## Features
The site offers resources, including basic and extended guides for systems administration, web development, gists and user guides on [some projects](/docs/repo). Additionally, it features articles about [space](/cosmos), too!

## Tech Specs

Designed and maintained by  [**@mkeithX**](https://github.com/mkeithx), the website is built using  [**Docusaurus**](https://docusaurus.io/), a popular static-site generator known for its ease of use and flexibility. The site’s source code is hosted on **GitHub**, allowing for collaborative development and version control. It is also hosted on [**Cloudflare**](https://developers.cloudflare.com/pages)., ensuring fast and reliable performance with enhanced security features.
