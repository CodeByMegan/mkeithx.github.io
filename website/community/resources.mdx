## Search

### [Algolia](https://algolia.com/)

Docusaurus has official support for Algolia DocSearch. The service is free for any developer documentation or technical blog. Read through the [checklist](https://docsearch.algolia.com/docs/who-can-apply/) and apply to the DocSearch program.


## Design

### [Infima](https://infima.dev/)

A minimalist CSS framework for rendering React components.

### [Shields.io](https://shields.io)

A service for concise, consistent, and legible badges.

### [Font Awesome](https://fontawesome.com/)

A font and icon toolkit based on CSS and LESS.

## Infrastructure and Deployment

### [GitHub Pages](https://pages.github.com/)

GitHub Pages is a static site hosting service designed to host your personal, organization, or project pages directly from a GitHub repository.

### [Cloudflare](https://cloudflare.com/)

Cloudflare offers various services including CDN, DDoS protection, and security.

### [Vercel](https://vercel.com/)

Vercel is a cloud platform for static sites and serverless functions that enables developers to host websites and web services that deploy instantly.
